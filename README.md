(種植篇)當你興高采烈打開網站，發現裡面一片荒涼，寫著「這片區域還是一片荒地」... 😱別緊張！這是因為你還沒告訴 Mindomo 哪些檔案要「種」出來。怎麼種花 (新增標籤)？回到你的 Mindomo 網站。打開你想分享的心智圖。找到 「屬性」 或 「描述」 (Description/Note) 的欄位。在裡面打上 #share 這個標籤。一定要存檔喔！👉 詳細圖文教學：Mindomo
