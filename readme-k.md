1. 🎨 **Mindomo 帳號** - 用來繪製心智圖的畫布

## ⚙️ 花園設定指南

### 第一步：取得你的花園鑰匙 🔑

1. 打開教學：[如何取得 Mindomo API](https://2blog.ilc.edu.tw/linkc/2025/11/21/mindomo-%e5%a6%82%e4%bd%95%e5%8f%96api/)
2. 跟著步驟取得你的 API 金鑰（這是進入花園的鑰匙）

### 第二步：客製化你的花園 ✏️

用文字編輯器打開 `mindomo_view.html`，調整以下設定：

#### 1️⃣ 設定花園入口的傳送門

找到這一行：
```javascript
const portalUrl = "https://你的網址.com";
```
這個傳送門可以連回：
- 你的個人首頁（例如：`https://gavintux.github.io/yi/`）
- 學校首頁
- 任何你想讓訪客前往的地方

#### 2️⃣ 安裝花園鑰匙

找到這一行：
```javascript
const apiKey = "你的API金鑰";
```
把剛才取得的 Mindomo API 金鑰貼上

#### 3️⃣ 設定花園管理員密碼 🔒

找到這一行：
```javascript
const adminPassword = "123456";
```
⚠️ **重要！** 請把 `123456` 改成只有你知道的密碼，保護你的私密想法！

#### 4️⃣ 更換花園招牌 🖼️

找到這一行：
```html
<img src="你的logo網址" alt="花園Logo">
```
上傳你喜歡的圖片作為花園標誌（可使用 imgur、Google Drive 等圖床服務）

## 🏷️ 重要：為你的想法加上標籤

### 🤔 為什麼我的花園是空的？

剛建好的花園是空的，因為你還沒有種下任何「公開」的想法種子！

### 🌱 如何種下想法種子：

1. 打開 Mindomo，選擇你想要分享的心智圖
2. 在心智圖設定中找到「標籤」功能
3. 加上 `#share` 標籤
4. 儲存後，這個想法就會出現在你的心智圖花園中！

**標籤的意義：**
- `#share` - 🌍 公開分享，任何人都能看到
- 沒有標籤 - 🔒 私密想法，只有管理員能看到
- 你可以自訂其他標籤來分類（例如：`#學習中` `#已完成`）

📚 **詳細教學**：[如何增加 Mindomo 標籤](https://2blog.ilc.edu.tw/linkc/2025/11/21/mindomo-%e5%a6%82%e4%bd%95%e5%a2%9e%e5%8a%a0%e6%a8%99%e7%b1%a4/)

## 💐 分享你的花束

### 什麼是花束？

花束就是把幾張相關的心智圖「綁」在一起，形成一個主題集合！就像真實的花束一樣，把最美的花朵組合起來分享給別人。

### 🎁 如何製作和分享花束：

#### 方法一：使用標籤組合
1. 為相關的心智圖加上相同的主題標籤
   - 例如：`#數學學習` `#科學實驗` `#閱讀心得`
2. 這些心智圖會自動在花園中群組顯示
3. 分享時可以說：「來看我的數學學習花束！」

#### 方法二：直接分享心智圖連結
1. 在你的心智圖花園中找到想分享的心智圖
2. 點擊心智圖，會顯示 Mindomo 的嵌入視圖
3. 複製該心智圖的網址
4. 分享給朋友，說：「這是我今天的學習花束！」

#### 方法三：分享整座花園
1. 複製你的花園網址（例如：`https://你的名字.github.io/專案名稱/`）
2. 透過以下方式分享：
   - 📧 Email 給老師或同學
   - 💬 在社群媒體上分享
   - 📱 傳訊息給朋友
   - 🎨 加入個人作品集網站
   
#### 方法四：嵌入到其他網站
如果你有自己的部落格或網站，可以：
1. 在心智圖花園中選擇心智圖
2. 取得 Mindomo 的嵌入程式碼
3. 放到你的網站中，讓訪客直接看到你的花束

### 💡 分享花束的創意點子：

- 📖 **學習筆記花束**：把一週的學習心智圖整理成花束
- 🎨 **創意發想花束**：把腦力激盪的想法組合起來
- 📚 **閱讀心得花束**：把讀過的書心智圖集合分享
- 🔬 **專題研究花束**：把專題的各個面向整理成花束
- 🌈 **每月精選花束**：定期挑選最棒的心智圖分享

### 📢 分享花束的最佳實踐：

1. **加上說明**：告訴別人這束花的主題是什麼
2. **定期更新**：讓你的花束保持新鮮
3. **回應互動**：如果有人給你回饋，記得回應
4. **標示日期**：註明花束的製作時間
5. **保持真實**：分享真實的學習過程，不用追求完美

## 🎮 照顧你的心智圖花園

### 👥 訪客模式（公開花園）
- 任何人都能來參觀你的花園
- 只會看到有 `#share` 標籤的心智圖
- 可以欣賞你的想法，但不能修改

### 🌱 園丁模式（管理員）
1. 點擊右上角 **管理員登入** 按鈕
2. 輸入你設定的密碼
3. 現在可以看到：
   - ✅ 所有公開的想法（有 `#share` 標籤）
   - 🔒 正在萌芽的私密想法（沒有標籤）
   - 🌈 整座花園的全貌

### 🌳 花園管理技巧

- **定期澆水** 💧：持續更新和完善你的心智圖
- **修剪枝葉** ✂️：移除過時或不需要的內容
- **播種新想法** 🌱：不斷添加新的心智圖
- **建立連結** 🔗：在心智圖之間建立關聯
- **整理花束** 💐：把相關的心智圖用標籤組織起來

## 🐛 花園常見問題

### 問題 1：花園打不開 🌵
- 檢查 GitHub Pages 是否已啟用
- 等待 5-10 分鐘讓種子發芽（設定生效時間）
- 確認網址是否正確

### 問題 2：無法連接 Mindomo API 🥀
- 確認 API 金鑰是否正確複製貼上
- 檢查 Mindomo 帳號是否正常
- 確認網路連線正常

### 問題 3：花園裡沒有植物（心智圖）🍂
- 確認心智圖有加上 `#share` 標籤
- 重新整理網頁（按 F5 或 Ctrl+R）
- 檢查瀏覽器的 Console（按 F12）看是否有錯誤訊息

### 問題 4：想分享特定的心智圖花束 💐
- 使用相同的標籤將心智圖分組
- 或直接分享整個花園的網址
- 可在社群媒體或訊息中附上花園連結
